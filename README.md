# BigMacIndex
Spring Boot tool to show in realtime the BigMac index, for comparing exchange types between two currencies. Allows input from user of current Big Mac prices in countries, and compares to currency exchange rate from online services.
#Characteristics
- Spring boot
- PostgreSql and Data JPA
- Maven
- Code checks with SonarLint
- Integration tests.
